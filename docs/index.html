Matthew Strimas-Mackey, Alison Johnston, Wesley M. Hochachka, Viviana Ruiz-Gutierrez, Orin J. Robinson, Eliot T. Miller, Tom Auer, Steve Kelling, Daniel Fink

2019-06-22

Welcome

These lessons comprise Part I of a workshop on best practices for using eBird data. This workshop covers extracting and processing eBird data with R, while Part II of the workshop covers using these data to model species distributions. The content of this workshop is also covered in the free online book eBird Best Practices.
